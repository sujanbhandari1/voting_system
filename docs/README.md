# Documentation

- Main report: `docs/Project_Documentation.md`

## How to use the provided Word template

If you need the final output as a `.docx` that matches `Document_Format.docx`, the simplest workflow is:

1. Open `Document_Format.docx` in Microsoft Word.
2. Copy content section-by-section from `docs/Project_Documentation.md` into the matching headings in the template.
3. Keep the same Heading styles (Heading 1/2) so Word can generate Table of Contents, List of Figures, and List of Tables.

## Optional: automatic export (if you install Pandoc)

If Pandoc is available on your machine, you can export the report to `.docx` and keep the template styling using `--reference-doc`.
